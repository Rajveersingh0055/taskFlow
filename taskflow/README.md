# TaskFlow

Phase 1 MERN stack setup for TaskFlow.

## Structure

```text
taskflow/
  client/
  server/
```

## Install

```bash
cd taskflow/client
npm install

cd ../server
npm install
```

## Run

```bash
cd taskflow/client
npm run dev
```

```bash
cd taskflow/server
npm run dev
```

## Environment

Copy `server/.env.example` to `server/.env` and fill in your values.
